My-Code-Card  -- A Chrome Extension for Code Note

Import this extension and select codes, right click to show contextmenu, chose "複製到Code Note ". Then click the icon extension icon show edit interface. Check language type, describe the content and update your note. If you want to review your note, review page support select though language or description.

Add option in contextmenu -- Chrome Extension API contextMenus 與 activeTab
Edit code -- Codemirror this JavaScript package can Highlight codes we edit 
Saving code -- Chrome Extension API storage, all codes will save at the same local storage
