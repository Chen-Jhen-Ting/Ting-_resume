Simpub  -- 線上編輯電子書與銷售平台

使用者能在此平台以讀者及作者身份操作，讀者可以搜尋與購買喜歡的電子書，購買電子書時，能依據對作者的喜好增加金額表示贊助，目前支援線上閱讀與下載PDF，以作者身份使用可以快速建立書本，新增封面，自由增減章節，以Markdown語法編輯，書本完成後可快速上架，編輯書本時有定時儲存功能不用擔心意外導致損失

負責範圍

  • 前期
    • User Story -- 發散列舉主次功能並考量製作時間縮限
    • Wireframe -- 使用 Figma 繪製網站所需頁面供組員參考
  • 中期
    • 書本線上編輯 -- 使用 Markdown 語法編寫書本，轉換成 HTML 供線上閱讀
    • 書本轉PDF -- 將線上編寫的書本轉成 PDF 並自動傳到 AWS-S3
    • AWS-S3 -- 使用 AWS Ruby SDK 將作者編輯的內容更新至S3
  • 後期
    • UIUX -- 依照讀者與作者身份使用網站的體驗給予建議
    • Monkey test -- 手動測試網站每一個功能與畫面
    • 前端畫面
