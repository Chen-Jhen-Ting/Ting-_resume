My-Code-Card  -- 紀錄程式碼的 Chrome Extension 筆記

使用者匯入套件後將程式碼選取，點右鍵顯示操作選項，選“複製到 Code Note”，再點 Code Note icon 顯示編輯頁面，選好語言類別，描述內容後新增內容，即可在筆記頁面搜尋與瀏覽已紀錄的筆記。

  • 增加 contextmenu 操作選項 -- Chrome Extension API contextMenus 與 activeTab
  • 編輯程式碼 -- Codemirror ， 此 JavaScript 套件能使程式碼 Highlight 與編輯
  • 儲存程式碼 -- Chrome Extension API storage，程式碼儲存到相同 local storage
