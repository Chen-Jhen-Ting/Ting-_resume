Simpub  -- 線上編輯電子書與銷售平台

此平台，能搜尋購買喜歡的電子書，能增加書本購買金額表示贊助，支援線上閱讀與下載 PDF ，作者可以快速建立書本，新增封面，自由增減章節，以 Markdown 語法編輯，書本完成後可快速上架，編輯書本時有定時儲存不用擔心意外導致損失

負責範圍

    • 書本線上編輯 -- 使用 Markdown 語法編寫書本，轉換成 HTML 供線上閱讀
    • 書本轉PDF -- 將線上編寫的書本轉成 PDF 並自動傳到 AWS-S3
    • AWS-S3 -- 使用 AWS Ruby SDK 將作者編輯的內容更新至S3
